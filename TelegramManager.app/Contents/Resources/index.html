🏷 Set App / Dock Name

This renames the Telegram.app bundle inside the account folder so each instance shows its own name in the Dock, Cmd+Tab switcher, and Finder — instead of all showing "Telegram".
Leave blank to reset to the account folder name.

⚠ The app bundle will be renamed on disk. Any running instance of this account must be restarted for the Dock change to take effect.
